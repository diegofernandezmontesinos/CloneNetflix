Frontend:

1) Clone the repository 
 git clone https://github.com/diegofernandezmontesinos/CloneNetflix.git

2) cd CloneNetflix
    npm install

3) install sass
    npm install -D sass   

4) install React router:
    npm i -D react-router-dom@latest    

 5) install axios for the data fetching
    npm i axios
